# Home Assistant App: Pocket ID

Home Assistant app for passkey authentication via OIDC based on [Pocket ID](https://pocket-id.org/).

## About

Pocket ID is a simple **OIDC provider** for **passkey authentication**. This app allows you to integrate passwordless login using WebAuthn/FIDO2, providing a seamless and secure authentication experience.
